```python
    # ── Conditional Auto-Refresh (30s sync when tasks running) ────
    if not is_demo:
        has_running = False
        if not projects_df.empty and "timestamps_log" in projects_df.columns:
            for _, row in projects_df.iterrows():
                log = row.get("timestamps_log", "")
                if isinstance(log, str) and get_status_from_log(log) == "running":
                    has_running = True
                    break

        _auto_interval = timedelta(seconds=30) if has_running else None

        @st.fragment(run_every=_auto_interval)
        def _auto_refresh():
            if has_running:
                st.rerun()

        _auto_refresh()
```

**How this works:**
- `has_running` is computed from projects_df (already loaded from Sheets) by checking each task's timestamps_log for running status
- `run_every` is dynamically set: 30s when running, None when idle (no fragment re-execution = no API calls)
- The fragment function is defined inside render_workspace, capturing `has_running` in its closure
- When the fragment fires after 30s, `st.rerun()` triggers a FULL app rerun — workspace reloads data from Sheets, rebuilds session state, re-renders all cards with fresh data
- When all tasks stop, the next rerun evaluates `has_running=False`, sets `run_every=None`, disabling the fragment

**Important:** Do NOT place any Sheets API calls inside the fragment. The fragment only calls `st.rerun()` — the actual data refresh happens during the full rerun in the normal data loading flow (lines 42-49).
